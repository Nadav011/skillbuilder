## Validation Protocol

Each gate MUST pass verification before skill approval.

**Scoring**:
- 70/70: GOD-TIER skill
- 49-51/52: Excellent (minor improvements needed)
- 45-48/52: Good (refinement required)
- < 45/52: REJECT (major issues)

**Critical Gates** (must ALWAYS pass):
- Gate 1 (Identity)
- Gate 8 (Zero Trust)
- Gate 10 (Type Safety)
- Gate 12 (APEX Laws)
- Gate 19 (Error Management)
- Gate 50-52 (Responsive/RTL compliance)
