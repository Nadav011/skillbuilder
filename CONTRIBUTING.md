# Contributing

Thank you for considering a contribution to SINGULARITY FORGE.

## English

### How to Contribute
- Open an issue with a clear description and reproduction steps.
- Propose changes via a pull request with a focused scope.
- Keep updates aligned with the existing gate and validation philosophy.
- Include documentation updates when behavior changes.

### Development Notes
- Validate skills with `./scripts/validate-skill.sh` where relevant.
- Keep structure consistent with `skill.yaml` and `SKILL.md` expectations.

---

## עברית

### איך תורמים
- פתח Issue עם תיאור ברור ושלבי שחזור.
- שלח Pull Request ממוקד ומוגדר היטב.
- שמור על התאמה לגישת האימות והשערים הקיימת.
- עדכן תיעוד כשמתבצע שינוי התנהגותי.

### הערות פיתוח
- יש להריץ `./scripts/validate-skill.sh` בעת הצורך.
- שמור על מבנה עקבי בהתאם ל־`skill.yaml` ו־`SKILL.md`.
